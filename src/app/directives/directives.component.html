Directive Concepts

Component Directives | Structural Directives | Attribute Directives

Component Directive : A component directive in Angular defines a new HTML element with associated behavior, template, and styles. It encapsulates reusable logic for modularity and maintainability.

Structural Directive : A structural directive in Angular alters the DOM layout by adding, removing, or manipulating elements. Examples include *ngIf, *ngFor, and *ngSwitch.

Attribute Directive : An attribute directive in Angular changes the appearance or behavior of an element. Examples include ngClass and ngStyle, which dynamically adjust classes and styles.
